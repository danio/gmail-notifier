gmail notifier

For using on windows python, PyQt4 and feedparser are required.

Windows installers for python may be downloaded from http://www.python.org/download

Windows installers for PyQt4 may be downloaded from http://www.riverbankcomputing.co.uk/software/pyqt/download

feedparser is currently (Feb 2010) distributed as a zip file which can be downloaded from http://code.google.com/p/feedparser/downloads/list. This file should be unzipped and then from a command prompt within the uncompressed directory run:
>setup.py install
to install it.
